# 인용문(BlockQuote)

```html
> 남의 말이나 글에서 직접 또는 간접으로 따온 문장.<br />
> (네이버 국어 사전)
```

> 남의 말이나 글에서 직접 또는 간접으로 따온 문장.<br/>
> (네이버 국어 사전)
