# 인용문(BlockQuote)

```html
> 남의 말이나 글에서 직접 또는 간접으로 따온 문장.<br />
> (네이버 국어 사전)
```

> 남의 말이나 글에서 직접 또는 간접으로 따온 문장.<br/>
> (네이버 국어 사전)

# 표(Table)

position 속성

| 값       |       의미        | 기본값 |
| -------- | :---------------: | -----: |
| static   |     기준 없음     |      O |
| relative |     요소 자신     |      X |
| absolute | 위치 상 부모 요소 |      X |
| fixed    |      뷰포트       |      X |

```html
값 | 의미 | 기본값 --|:--:|--: static | 기준 없음 | O relative | 요소 자신 | X
absolute | 위치 상 부모 요소 | X fixed | 뷰포트 | X
```
